🔧 Firebase 設定
填入你的 Firebase 設定，資料才能在薯和豚的裝置之間即時同步。
1. 前往 console.firebase.google.com
2. 建立專案 → Realtime Database → 建立資料庫
3. 選亞洲地區、以測試模式開始
4. 規則頁面確認 read/write 皆為 true
5. 專案設定 → 一般 → 新增網頁應用程式 → 複製各欄位
